- Install:

	>> mcolon_install

- Try:

	>> mcolon([0 10],[1 0.5],[2 12])

	ans =
	
        	 0    1.0000    2.0000   10.0000   10.5000   11.0000   11.5000   12.0000


- Overload colon operator:

	>> mcolonops('mcolon')
	>> 'a0A':'z9Z'

	ans =
	
		abcdefghijklmnopqrstuvwxyz0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ
